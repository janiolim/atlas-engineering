# Atlas

Portfólio técnico focado em engenharia elétrica, física aplicada e sistemas embarcados.

## Estrutura

- `/works` → projetos de engenharia e sistemas
- `/blog` → notas técnicas e pesquisa
- `/about` → perfil e trajetória

## Tecnologias

- Astro
- MDX
- Content Collections

## Objetivo

Organizar aprendizado técnico em forma de projetos reais, experimentos e documentação estruturada.
