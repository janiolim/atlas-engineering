# Astro Keel

A minimal, neutral, and modern portfolio and blog theme for Astro. Astro Keel pairs a calm neutral base with a single configurable accent color, generous whitespace, and clean typography — built on Astro 7 with Content Collections, tags, RSS, and first-class dark mode.

> **Keel** — the structural backbone of a ship. The name reflects the design intent: stripped of ornament, all structure and spine.
